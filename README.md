# SpareTimeGizmos Web Site Backup

This project is an attempted replication of the SpareTimeGizmos.com web site originally created and maintained by Bob Armstrong.

The goal is to create a git repository that will reflect the original web site on a github.io URL such as:

[https://bobkuczewski.github.io/SpareTimeGizmosSite/index.htm](https://bobkuczewski.github.io/SpareTimeGizmosSite/index.htm)

Here's an image of the home page as it currently appears:

![SpareTimeGizmosSite Home Page](SpareTimeGizmosSite_GitHub.png)
